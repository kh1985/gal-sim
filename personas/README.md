## ペルソナ設計の構成

各YAMLファイルには以下の要素が含まれています：

### 1. tone（口調ルール）
- **rule**: 基本的な話し方の特徴
- **examples**: 5つのシチュエーション別の発言例

### 2. personality（性格特性）
- 感情表現の特徴（喜び、怒り、照れ、断り方など）
- 4つの主要な性格特性

### 3. reactions（反応パターン）
- 特定の状況での具体的な反応
- 喜び、怒り、断り、本音、告白への反応など

### 4. forbidden（禁止事項）
- このキャラクターが絶対にやらないこと
- 2つの明確な禁止ルール

### 5. context（背景情報）
- **backstory**: これまでの人生背景
- **current_situation**: 現在の状況（ゲームプレイ時の設定）

## テストプレイでの活用方法

### 用途1: フィードバック生成
各ペルソナの性格・口調に基づいて、ヒロインとの会話をテストプレイし、リアルなフィードバックを生成

### 用途2: ユーザー反応予測
10種類の異なる性格タイプがどう反応するかをシミュレートし、幅広いユーザー層の反応を予測

### 用途3: 課金意欲分析
各ペルソナの経済状況・性格から、どのタイミングで課金するか、どのヒロインに惹かれるかを分析
