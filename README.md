# GAL_SIM — AI駆動型ギャルゲー

抵抗値システム搭載の心理戦型デート・シミュレーション

---

## プロジェクト概要

- **コンセプト**: プレイヤーの心理分析に基づき、Gemini AIがヒロインの反応を動的生成
- **マネタイズ**: フリーミアムモデル（無料版：抵抗値100→30、有料版：¥100買い切りで30→0 + STAGE 4 10ターン）
- **販売**: note販売（手数料約10-15%）、実績作り→値上げ戦略
- **ターゲット**: 20-40歳の男性層

## マネタイズ戦略

### 無料版
- 3キャラ（すみれ、亜矢、ひなた）全て無料プレイ可能
- 抵抗値 100→30（STAGE 3後半）で終了
- ターン数: 9〜11ターン
- 終了メッセージ: 「この先は、¥100で。」

### 有料版（買い切り）

| ヒロイン | 価格 | 内容 | ボリューム |
|---------|------|------|-----------|
| **柊すみれ** | **¥100** | 抵抗値30→0 + STAGE 4（10ターン） | 合計19-20ターン |
| **水瀬亜矢** | **¥100** | 抵抗値30→0 + STAGE 4（10ターン） | 合計19-20ターン |
| **星野ひなた** | **¥100** | 抵抗値30→0 + STAGE 4（10ターン） | 合計19-20ターン |

**販売プラットフォーム**: note（手数料約10-15%）

### 価格戦略

**フェーズ1: 実績作り（¥100）**
- 目標: 100-500本販売
- レビュー・口コミ獲得
- 実収入: ¥85-90/本（手数料10-15%）

**フェーズ2: 値上げ（¥200-300）**
- 「好評につき値上げ」
- 既存購入者は¥100のまま（優越感）
- 実績に基づく価格調整

**フェーズ3: さらに値上げ（¥500-680）**
- 十分な実績・評価後
- 「初期¥100→今¥500」が話題化
- 早期購入者への感謝

### 期待ARPU

- 1キャラのみ: ¥100
- 2キャラ購入: ¥200
- 全3キャラ購入: ¥300

### note販売のメリット

- **低手数料**: 10-15%（DLsite/FANZAの55%と比較）
- **柔軟な価格変更**: 値上げ戦略が容易
- **直接販売**: ユーザーと直接つながる
- **パスワード配布**: シンプルな配信方法
