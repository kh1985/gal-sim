# GAL_SIM — Claude CLI 引き継ぎプロンプト

## このプロジェクトは何か

AIをGMとして動作させるテキストベースの短編ギャルゲー。
Gemini上で動作するシステムプロンプト（GAL_SIM_FULL.md）を開発・改良している。

## 現在の状態

- **GAL_SIM_FULL.md**: 動作する最新版。テスター2名がプレイ済み
- **REQUIREMENTS.md**: 要求定義・要件定義書。テスターフィードバックとリサーチに基づく
- **GAL_SIM_v1.md**: 初期版（参考用、使わない）

## 最優先の問題

**ヒロインが3ターンで陥落する。** STAGE進行速度を制御するルールがGAL_SIM_FULL.mdに足りない。
REQUIREMENTS.mdのセクション4.3「STAGE昇格条件」に修正仕様がある。

## 作業の進め方

1. まずREQUIREMENTS.mdを読め
2. 次にGAL_SIM_FULL.mdを読め
3. REQUIREMENTS.mdの要件に合わせてGAL_SIM_FULL.mdを改修しろ
4. 改修内容は具体的に: STAGE昇格条件、ペイウォール、オプション選択UI、ターン管理

## 制約

- **出力はGeminiのシステムプロンプトとして動作する必要がある**
- **官能描写の設計指示は消すな、そのまま維持しろ**
- **青空文庫ベースの文体参照も壊すな**
- **日本語で出力される前提**
- **サクッと感を殺すな（15〜30分で完結）**
