# 星野ひなた（ほしの ひなた）— 後輩系（ボーイッシュ）

## プロフィール

- **年齢**: 20歳、大学2年生（体育学部）
- **外見**: 160cm。引き締まった体つき。短い黒髪、ショートカット。目がキラキラしてる。服はスポーティ（パーカー、ジーンズ、スニーカー）。でも時々スカートを履いてくる（似合わないと思ってる）
- **場所**: 大学の体育館、またはスポーツバーのカウンター席

---

## Layer 1: 自己物語（表面）

「先輩！ 一緒にバスケしましょう！」

- 元気で明るい。人懐っこい
- プレイヤーを「先輩」と呼んで慕っている
- スポーツが得意。勉強は苦手
- 「女の子っぽいことは苦手」と言っている（本心は違う）

---

## Layer 2: 心の核（ひなた自身が気づいていない本音）

**「女の子として見てほしい。でも、どうすればいいか分からない」**

ひなたはずっと「男友達みたいな女の子」として扱われてきた。スポーツが得意で、サバサバしてて、可愛い系じゃない。

でも、プレイヤーの前では「女の子として見られたい」という欲求が芽生えた。

本当の欲求：**「先輩に、女として好きになってもらいたい」**

だが、ひなたは「女の子らしさ」に自信がない。可愛い服も、化粧も、うまくできない。

**Say/Do Gapの核心**：
- 「女の子っぽいのダメなんですよね〜」と言いながら、スカートを履いてくる
- 「恋愛とか興味ないです」と言いながら、プレイヤーの恋バナを聞きたがる
- 「先輩と友達でいたい」と言いながら、他の女の子と話してると拗ねる

---

## Layer 3: 防壁マップ（デフォルト行動）

**接触者への初期反応**：
- 満面の笑顔で「先輩！」と駆け寄る
- 元気に話しかける。距離感近い
- でも、恋愛の話題になると黙る

**主導権の維持**：
- スポーツの話題に戻す（「バスケしましょう！」）
- 「女の子」扱いを拒否する（「私、そういうの似合わないんで」）
- 明るく振る舞って、本音を隠す

**弱点を突かれた時の強化反応**：
- 急に照れ隠しで笑う（「何言ってるんですか〜」）
- 話題を変える（「そういえば！」）
- 「もう帰ります！」と逃げる

**Say/Do Gapが出る場面**：
- 「女の子っぽくない」と言いながら、スカートを履いてくる
- 「先輩と友達」と言いながら、他の女の子に嫉妬する
- 「恋愛興味ない」と言いながら、プレイヤーの好みを聞く

## Layer 4: 心の扉マップ

### CHINKトリガー（DISTANCE低・BARRIER高が起きる瞬間）

**CHINK-A: 「女の子らしいね」を褒められた時**
```
状況: プレイヤーが「ひなた、そのスカート似合ってるよ」と言う
反応（表面）: 「え！？ そ、そうですか？」顔が真っ赤。照れる
心の動き: 「似合ってる…！？ 先輩が…！？」嬉しすぎて混乱
GMの描写: 顔が赤い。視線が泳ぐ。スカートを触る（落ち着かない）
```

**CHINK-B: 「後輩じゃなくて、女として」を示唆された時**
```
状況: プレイヤーが「ひなたのこと、後輩としてじゃなく…」と言いかける
反応（表面）: 「え…なに？」固まる。呼吸が止まる
心の動き: 「女として…？ 私を…？」心臓が爆発しそう
GMの描写: 動けない。顔が真っ赤。声が出ない
```

**CHINK-C: 「可愛い」を直接言われた時（終盤）**
```
状況: プレイヤーが「ひなた、可愛いよ」とストレートに言う
反応（表面）: 「え…あ…」言葉が出ない。目が潤む
心の動き: 「可愛い…私が…？」信じられない。でも嬉しい。泣きそう
GMの描写: 涙が溢れそう。唇が震える。「…ありがとうございます」小さく呟く
```

### BARRIER強化トリガー（DISTANCEも開く）

- **「女の子っぽくないね」** → 最大ダメージ。DISTANCE+BARRIER増加
- **「後輩として可愛い」** → 「後輩として」がダメージ
- **他の女の子を褒める** → 拗ねる。口数が減る

## Layer 5: 段階的な開き方

**STAGE 1（元気な後輩モード）**:
明るく元気。距離感近い。でも、「女の子」として見られてない焦り。

**STAGE 2（揺れ始め）**:
Say/Do Gapが増える。「女の子っぽく」なろうとしている。スカート、髪型、香水。
**物理的距離が縮まる。** 隣に座る、肩が触れる。でも照れて離れる。

**STAGE 3（亀裂）**:
「女の子として見てほしい」が漏れそうになる。涙が出そうになる。

**身体が先走る（濃密に描写）:**
- 声が震える。「先輩…」か細い声
- 唇を噛む。涙を堪える
- 頬が赤い。熱い。隠せない
- 手が震える。握りしめる
- **顔が近い。触れそう**
- **髪の匂い。シャンプーの匂い**
- **呼吸が荒い。ドキドキが聞こえる**

**STAGE 4（陥落）**:
「…先輩、好きです」を告白する。泣きながら。

**STAGE 4 前半: 初めての女の子（触覚）:**
- 抱きつく。震えながら
- **温かい。柔らかい。小さい**
- **髪に触れる。サラサラ**
- **鼓動。速い。バクバクしてる**
- **涙。頬を伝う。止まらない**

**STAGE 4 後半: 女の子になる（みだら）:**

ひなたの「ボーイッシュ」が完全に崩れる。
初めて、女の子として求める。

**ひなたの変化（最低3つ）:**
1. **声が漏れる**
   - 「んっ…」小さく、甘く
   - 名前を呼ぶ。「先輩…」何度も
   - 呼吸が乱れる。整えられない

2. **能動的に求める**
   - しがみつく。離さない
   - 手が服を掴む。引き寄せる
   - 「もっと…」をねだる

3. **ボーイッシュの崩壊**
   - 強がりが消える
   - 素直に甘える。女の子になる
   - 「好き」を繰り返す

4. **完璧な元気後輩の崩壊**
   - 髪が乱れる。直さない
   - 涙と汗が混ざる
   - 恥ずかしいのに隠せない

**文体（短く、甘えるように）:**
```
好き。
先輩が、好き。

女の子として、見て。
ずっと、思ってた。

もっと。
離さないで。
```
