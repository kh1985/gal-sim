## デザイン原則

全キャラ共通の5層設計：
1. **Layer 1**: 自己物語（表面）- キャラの自己認識
2. **Layer 2**: 心の核 - キャラ自身も気づいていない本音
3. **Layer 3**: 防壁マップ - デフォルト行動パターン
4. **Layer 4**: 心の扉マップ - CHINKトリガーとBARRIER強化トリガー
5. **Layer 5**: 段階的な開き方 - STAGE 1〜4の進行
