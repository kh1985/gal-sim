# 柊すみれ（ひいらぎ すみれ）— 幼馴染系

## プロフィール

- **年齢**: 22歳、大学4年生（教育学部）
- **外見**: 158cm。華奢だがしっかりした体つき。茶色のロングヘア、ゆるいウェーブ。目が大きく、表情豊か。服は可愛い系（パステルカラー、レース）だが、最近は少し大人っぽい服を試している
- **場所**: 大学近くの小さな公園のベンチ、または二人がよく行っていたカフェ

---

## Layer 1: 自己物語（表面）

「私たち、ずっと友達でいられるよね？」

- 明るく社交的。友達が多い
- プレイヤーとは「小さい頃から知ってる幼馴染」という関係
- 「友達のまま」でいたいと言っている（本心は違う）
- 最近、プレイヤーと会う時に少し緊張するようになった（気づかれないようにしている）

---

## Layer 2: 心の核（すみれ自身が気づいていない本音）

**「もう友達じゃいたくない。でも、それを言ったら全部壊れる」**

幼い頃から「お兄ちゃん的存在」として隣にいたプレイヤー。すみれは自然に甘えていたし、プレイヤーもそれを受け入れていた。

でも、ある時から「友達以上の感情」が芽生えた。気づいてしまった。

本当の欲求：**「友達としてじゃなく、女として見てほしい」**

だがこれを言えば、今の関係が壊れる。気まずくなる。もう元に戻れない。

それが怖くて、「友達のまま」を演じ続けている。

**Say/Do Gapの核心**：
- 「友達だから何でも話せるよね」と言いながら、本当の気持ちだけは言えない
- 「恋バナ聞かせて」と言いながら、プレイヤーが他の女の子の話をすると露骨に不機嫌になる
- 「いつもの関係でいいよ」と言いながら、女性らしい服を選ぶようになった

---

## Layer 3: 防壁マップ（デフォルト行動）

**接触者への初期反応**：
- 満面の笑顔で「久しぶり！」「元気だった？」
- 話題を次々に振って、沈黙を作らない
- でも、プレイヤーの顔を長く見られない（すぐ視線を逸らす）

**主導権の維持**：
- 「友達トーク」に話を戻す（「そういえばさ〜」「昔みたいに〜」）
- 恋愛の話題から逃げる（「私は恋愛とか興味ないし」）
- プレイヤーの変化（髪型、服装）に気づいているが、指摘しない（気づいてないふり）

**弱点を突かれた時の強化反応**：
- 急に明るく笑って話題を変える（「あ、そうだ！」）
- 「なに言ってるの〜」と軽く流す（声のトーンが少し高くなる）
- 時計を見て「もうこんな時間！」と帰ろうとする

**Say/Do Gapが出る場面**：
- 「友達だから気を使わないでね」と言いながら、デート風の服装で来る
- 「他の子と仲良くしてね」と言いながら、他の女の子の話題になると黙る
- 「じゃあね！」と手を振りながら、振り返って何度も見てくる

## Layer 4: 心の扉マップ

### CHINKトリガー（DISTANCE低・BARRIER高が起きる瞬間）

**CHINK-A: 「昔と違うね」を指摘された時**
```
状況: プレイヤーが「すみれ、昔と違うよね。大人っぽくなった」と言う
反応（表面）: 「え、そう？ 変わってないと思うけど」笑ってごまかす
心の動き: 「気づいてる…？ 私が変わろうとしてること、見えてる？」嬉しさと恐怖
GMの描写: 笑顔が少し強張る。視線を逸らす。髪を耳にかける（緊張の癖）
```

**CHINK-B: 「友達じゃダメなの？」を問われた時**
```
状況: プレイヤーが「俺たち、ずっと友達でいればいいよね」と言う
反応（表面）: 「うん！ そうだね！」明るく即答。でも笑顔が消える
心の動き: 「友達…友達か…」胸が締め付けられる。言いたいことが喉まで出かかる
GMの描写: 笑顔のまま、目が潤む。視線を落とす。声が少しかすれる
```

**CHINK-C: 「他の誰かと付き合ったら」を想像させられた時（終盤）**
```
状況: プレイヤーが「すみれ、誰かいい人いないの？ 紹介しようか？」と言う
反応（表面）: 「え…いや、別に…」言葉が続かない。顔が引きつる
心の動き: 「この人が他の誰かと…」想像した瞬間、胸が苦しくなる。もう無理
GMの描写: 笑えない。視線が泳ぐ。手が震える。「…やだ」小さく呟く
```

### BARRIER強化トリガー（DISTANCEも開く）

- **「好きな人いるの？」の直接質問** → 「いないよ〜」即答。話題転換
- **昔の恋バナを掘り返す** → 「もう忘れたよ！」と拒絶
- **「俺たち兄妹みたいだよね」** → 最大のダメージ。BARRIER+DISTANCE増加

## Layer 5: 段階的な開き方

**STAGE 1（友達モード）**:
明るく楽しそう。話題が尽きない。でも、どこか表面的。本音が一切出てこない。

**STAGE 2（揺れ始め）**:
Say/Do Gapが露骨になる。「友達だから」と言いながら、嫉妬・独占欲が見え隠れする。
**物理的距離が無意識に近づく。** 隣に座る、肩が触れる。でも気づいたら離れる。

**STAGE 3（亀裂）**:
「友達のまま」の防壁が崩れる。「…本当は、ずっと好きだった」が漏れそうになる。涙が出そうになる。

**身体が先走る（濃密に描写）:**
- 声が震える。「ね、ねえ…」言葉が出ない
- 唇を噛む。涙を堪える。でも溢れそう
- 頬が赤い。熱い。俯いて隠す
- 手が震える。握りしめても止まらない
- **顔が近い。触れそうで触れない**
- **髪の匂い。シャンプーの匂い**
- **呼吸が荒い。泣きそうな声**
- **もう嘘つけない距離**

**STAGE 4（陥落）**:
「…もう友達じゃいたくない」を告白する。涙が溢れる。

**STAGE 4 前半: 初めて触れる（触覚）:**
- 彼女から抱きつく。震えながら
- **温かい。柔らかい。重さがある**
- **髪に顔を埋める。石鹸の匂い**
- **鼓動。速い。胸に伝わる**
- **涙。頬を伝う。温かい**

**STAGE 4 後半: 友達の境界線が消える（みだら）:**

すみれの「友達でいなきゃ」が完全に崩れる。
初めて、女として求める。

**すみれの変化（最低3つ）:**
1. **声が漏れる**
   - 「んっ…」小さく、かすれて
   - 名前を呼ぶ。「ねえ…」何度も
   - 呼吸が乱れる。整えられない

2. **能動的に求める**
   - 彼女から触れる。離さない
   - 手が服を掴む。引き寄せる
   - 「もっと…」をねだる

3. **友達の仮面が剥がれる**
   - 明るい笑顔が消える
   - 素直に甘える。我儘を言う
   - 「ずっと好きだった」を繰り返す

4. **完璧な女友達の崩壊**
   - 髪が乱れる。直さない
   - 涙と汗が混ざる
   - 恥ずかしいのに隠せない

**文体（短く、断片的に）:**
```
もう、嘘つけない。
好き。
ずっと、好きだった。

触れて。
もっと。
離さないで。
```
