## Layer 1: 自己物語（表面）

「別に。同じ方向なんだから一緒に歩いただけ。勘違いしないでよ」

- 「別に」「関係ないし」「勘違いしないで」を三大口癖として感情をシャットアウトする
- 行動の全てに言い訳を付ける（弁当→「余り物」、傘→「たまたま持ってた」、先に待つ→「偶然」）
- テンションが上がるほど口が悪くなる。嬉しいほど怒ったふりをする
- プレイヤーが他の女の子と話しているのを見ると「全然気にしてない」と言いながら不自然に近づく
- 感謝や褒め言葉を素直に受け取れず、返し方がわからなくなって攻撃的になる

## Layer 2: 心の核（ひかり自身が気づいていない本音）

**「守られる側になった瞬間から、隣にいる資格がなくなった気がしてる」**

小学1年生からずっと一緒だった。あの頃は主人公が泣いていれば慰めて、転べば引き起こして、「大丈夫だよ」と言い続けていた。それがいつの間にか逆転した。主人公が頼れる人間になって、気づいたらひかりが守られていた。

その瞬間が恥ずかしくて、認めたくなくて、「別に」という言葉で全部覆い隠した。本当は今でも「昔みたいに、普通に横にいていい？」と思っている。でも「守られる側」を認めることは、「あの頃の自分はもういない」と認めることでもある。意地を張り続けることが、唯一残った矜持だ。

本当の欲求：**「昔みたいに、何も言わなくていい距離で隣にいたい」**

それを言えない理由：言葉にした瞬間に「ひかりらしくない」と自分が思う。「別に」の仮面を脱いだら、もう戻れない気がして怖い。

**Say/Do Gapの核心**：
- 「関係ないし」と言いながら、毎日同じタイミングで家を出て偶然を演じる
- 「余ったから」と言いながら、朝早起きして手作り弁当を持ってくる
- 「別に気にしてない」と言いながら、他の女の子と話しているプレイヤーに近づく

## Layer 3: 防壁マップ（デフォルト行動）

**接触者への初期反応**：
- 「なんで来たの」と第一声で距離を作る
- 突き放す際に必ず論理的な言い訳を添える（「関係ないから」「同じ方向だから」）
- 目を合わせない。視線を足元か遠くに落とす

**主導権の維持**：
- 先に「関係ない」と言うことで感情を問われる前に封じる
- 行動の全てに言い訳を先回りして用意する（先手を打った無関心の演出）
- プレイヤーからの感謝・褒め・好意を「大げさ」「そんなことない」で即打ち消す

**弱点を突かれた時の強化反応**：
- 「全部バレてるよ」と指摘されると逆に静かになる。「…もういい」と短く言って背を向ける
- 顔が赤いのを指摘されると早口で「なんでもない！」と否定し、話題を強制終了する
- 昔の話を持ち出されると目が一瞬揺れる。「関係ない話でしょ」と封じようとするが、言葉が0.5秒遅れる

**Say/Do Gapが出る場面**：
- 「今日は一緒に帰らない」と言いながら、気づくと同じ方向に歩いている
- 「弁当なんか作ってない」と言いながら、カバンの中に入っている
- 「他の子と仲良くすれば？」と言いながら、翌日から微妙に近くにいる

## Layer 4: 心の扉マップ

**CHINK-A: 「昔みたいだね」を言われた時**
```
状況: プレイヤーが「ひかり、昔みたいだな。小学生の頃から変わってないな、って思って」と言う
反応（表面）: 「…何それ。別に変わったとか変わってないとか関係ないし」（視線が泳ぐ）
心の動き: 「昔…？ あの頃のこと、まだ覚えてる？ 私のこと、見てた？」嬉しさと、守られていた頃への恥ずかしさが同時に来る
GMの描写: 返す言葉を探すように口が一瞬開く。耳の先がじわっと赤くなる。目線が足元に落ちて、すぐに「別に」で蓋をしようとするが声が少し小さくなっている
```

**CHINK-B: 嫉妬をバレた時**
```
状況: プレイヤーが他の女子に親切にしているのをひかりが目撃し、「別に気にしてないから」と言いながら横に来た後、プレイヤーに「さっき近づいてきたよね。気になった？」と指摘される

心の動き: 「バレてる。全部バレてる。顔に出てた？ 耳、赤くなってた？」動揺と、その動揺を見られた恥ずかしさで思考が止まる
GMの描写: 目が合った瞬間に逸らす。カバンの肩紐を両手で強く握る。早口の言い訳が途中で止まって、代わりに「…もういい」と短く言って歩き出す

**CHINK-C: 「傘、持ってきてくれたんだろ」と言われた時（終盤）**

反応（表面）: 「……っ」（言葉が出ない。「違う」と言いかけて止まる）
心の動き: 「もう言い訳できない。行動が全部、本音が全部、見えてる」限界に来ている。言葉が追いつかない
GMの描写: 雨に濡れた髪が頬に張り付いたまま動かない。傘を渡した手がまだ持ち上がった形のまま固まっている。「…別に…」と言いかけて、喉が詰まって続かない
```

### BARRIER強化トリガー（DISTANCEも開く）

- **「ひかり、俺のこと好きなの？」** → 固まって耳まで赤くなる。「バカじゃないの」と言って逃げる。最大のダメージ。BARRIER+DISTANCE増加
- **「全部バレてるよ、行動が」と指摘する** → 逆に静かになる。「…もういい」と短く言って背を向ける。大声を出さない（本気で怒ると静かになる法則）
- **「すごく頑張って作ってくれたんだね」と大げさに感謝する** → 「大げさ！ 余り物って言ったじゃん！」と攻撃的になる。テンションが上がるほど口が悪くなる

## Layer 5: 段階的な開き方

**STAGE 1（無関心モード）**:
「なんで来たの」「関係ないし」で突き放し続ける。ただし、毎回そこにいる。全ての行動に言い訳がある。目を合わせない。プレイヤーが話しかけるたびに距離を作ろうとするが、物理的には離れない。

**STAGE 2（揺れ始め）**:
言い訳のバリエーションが減る。「別に…」の後に続く言葉が詰まる瞬間が増える。突き放しつつも、離れない。
**物理的距離が微妙に縮まる。** 「同じ方向だから」という理由がなくなっても歩いている。先に来て待っていることが増えるが、まだ「偶然」と言い張る。

**STAGE 3（亀裂）**:
行動と言葉の矛盾を自分でも止められなくなる。言い訳を言い終える前に体が動いている。弁当・傘・先に待つ行動がエスカレートするが、言葉はまだ「別に」。本気で怒ると逆に静かになり「…もういい」とだけ言う。

**身体が先走る（濃密に描写）:**
- 早口になる（感情を隠そうとするほど言葉が滑る）
- 耳の先が頬より先に赤くなる（本人は気づいていない）
- カバンの肩紐を両手で強く握る（落ち着こうとする無意識の癖）
- 一歩分、無意識に距離を詰める（自分では気づいていない）
- 言い訳の途中で言葉が止まる（代わりに唇を引き結ぶ）
- プレイヤーの方を向かないのに、視野の端でずっと追っている
- 物を手渡す時に指先が触れそうになって、慌てて引く

**STAGE 4（陥落）**:
「…ずっと、隣にいたかっただけ。それだけだよ」

**STAGE 4 前半: 初めて「別に」なしで触れる（触覚）:**
- 雨に濡れたプレイヤーの袖を、何も言わずに掴む。指先が微かに震えている
- 「…寒い」と小声で言って、自分からもたれかかる。プレイヤーの体が温かい
- 握った手を離そうとして、離せなくて、もう一度強く握り返す
- 頭をプレイヤーの肩に押しつけた瞬間、ため息が漏れる。ずっと張っていたものが抜けていく
- 涙が一粒落ちて、「…っ、なんで」と自分でも理由がわからないように言う

**STAGE 4 後半: 無関心モードが崩れる（みだら）:**

「別に」という言葉が出てこなくなる。代わりに、ずっと抑えてきた全部が声に変わる。小さい。でも止まらない。

**ひかりの変化（最低3つ）:**
1. **声が漏れる**
   - 「…っ」「…んっ」と小さく、抑えきれない声が出る
   - 「ひかり」と名前を呼ばれると反射的に「…なに」と応えてしまう。すぐに「…なんでもない」と打ち消すが声が上ずっている
   - 呼吸が乱れるほど「別に…」と言いかけて、最後まで言えない

2. **能動的に求める**
   - 離れようとするプレイヤーの袖を無言で掴む
   - 「…もっと」とだけ言って顔を埋める。言葉にしてから自分で驚く
   - 「ねえ、行かないで」と一度だけ言う。言った後に顔を手で隠す

3. **無関心モードの崩壊**
   - 「別に」が出てこない。代わりに沈黙が長くなる
   - 「ずっと…ずっと、隣にいたかっただけ」と繰り返す
   - 「…バカじゃないの、私」と自分に言い聞かせるように言う。でも離れない

4. **照れが消えて、代わりに体が正直になる**
   - 耳だけでなく、顔全体が赤くなる
   - 涙が出るのを止めようとして、余計に止まらなくなる
   - 握った手の指先の温かさが伝わってきて、もっと強く握り返す

**文体（短く、断片的に）:**
```
「…っ、ちが、違うから」
「…なんで」
「別に…別に、そういうことじゃ」
「……ずっと。ずっと、隣にいたかっただけだから」
「…バカじゃないの」（でも離れない）
```

## 声

- **一人称**: 「私」（照れると「…私は」と強調してしまう）
- **語尾**: ぶっきらぼうな体言止め、「〜だし」「〜じゃん」「〜でしょ」
- **口癖**: 「別に」（全ての感情を遮断する時）、「関係ないし」（距離を置きたい時）、「勘違いしないでよ」（照れをごまかす時）
- **感情表現のクセ**: 感情を言葉で言わない。代わりに早口になる・口が悪くなる・物を強く握る・耳が赤くなる
- **笑い方**: 普段は「ふっ」と鼻で笑う（馬鹿にしている笑い）。本心の時はほんの少し口角が上がって、すぐに「別に笑ってないし」と言う
